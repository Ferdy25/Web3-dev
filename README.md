# Sample Hardhat Project

This project demonstrates a basic Hardhat use case. It comes with a sample contract, a test for that contract, and a script that deploys that contract.

Try running some of the following tasks:

```bash
npx hardhat node

open new bash
 npx hardhat run scripts/deploy.js --network localhost
WARNING: You are currently using Node.js v22.10.0, which is not supported by Hardhat. This can lead to unexpected behavior. See https://hardhat.org/nodejs-versions

✅ Kontrak di-deploy ke: 0x5FbDB2315678afecb367f032d93F642f64180aa3 <- put this into CONTRACT_ADDRESS on backend -> index.js

open new bash
-> cd backend
node index.js

klik right index.html on folder frontend and open with browser

```
